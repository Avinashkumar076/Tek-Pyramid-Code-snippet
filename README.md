## 📂 How It Works

1. Add a JavaScript snippet as a new file or inside the relevant folder.
2. Write the question at the top.
3. Do not mention the answer immediately.
4. Add the correct output and explanation at the bottom.
5. Keep explanations short, clear, and beginner-friendly.
